# db-python

Eine einfache Library für die unter `http://reiseauskunft.bahn.de` erreichbare Schnittstelle der Bahn. Diese API wird u.a. für die "DB-Navigator"-App verwendet.

# Features

- Gültige Prüfsumme generieren 
- Suche nach Bahnhöfen
- Verbindungssuche
